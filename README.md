### Python Journey
Сборник практических задач и решений на языке Python.

#### 1. Описание
Этот репозиторий — лаборатория для экспериментов с кодом, решения задач разного уровня сложности и закрепления изученного материала.

#### 2. Как использовать
1. **Клонировать репозиторий:**
   `git clone https://github.com/marina88813/python-journey.git`
2. **Перейти в папку:**
   `cd python-journey` 
3. **Запустить нужный файл:**
   `python название_файла.py` (например, `python heart.py`)
